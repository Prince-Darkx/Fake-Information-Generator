## 🕵️‍♂️ **Fake Information Generator**

> *Generate realistic fake identities for testing, privacy, or just for fun.*  
> *Crafted with 🖤 by @Prince-Darkx*

---

### 📌 **Features**
- 🎭 **Two reliable API sources** – `randomuser.me` & `namefake.com`  
- 🔄 **Automatic fallback** – if one API fails, the next is tried  
- 🌍 **Indonesian‑focused** – names, addresses, and phone numbers  
- 📱 **User‑friendly loop** – generate as many as you want, exit anytime  
- 🖥️ **Cross‑platform** – works on Windows, Linux, and macOS
